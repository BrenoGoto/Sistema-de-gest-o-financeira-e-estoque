Insanus Cellsystem - PDV & Gestão de Estoque

O **Insanus Cellsystem** é um sistema de Ponto de Venda (PDV) e gerenciamento comercial completo, desenvolvido para automatizar o fluxo de caixa, o controle de mercadorias e o registro de contas de clientes de forma ágil e segura.

 Funcionalidades Principais

- **Fluxo de Caixa Diário:** Controle rigoroso de entrada de dinheiro, faturamento e definição de troco inicial do dia.
- **Controle de Estoque Inteligente:** Validação em tempo real da disponibilidade de produtos antes de finalizar a venda, com baixa automática no banco de dados para evitar inconsistências.
- **Módulo de Clientes ("Fiado"):** Área exclusiva para cadastrar, registrar débitos e acompanhar o histórico de pagamentos de clientes mensalistas ou contas a prazo.
- **Relatórios e Histórico:** Filtragem avançada de vendas por métodos de pagamento (PIX, Cartão, Dinheiro) e integração de rotinas de backup.

 Tecnologias Utilizadas

O projeto adota a arquitetura Full-Stack:

- **Frontend:** HTML5, CSS3 (Interface moderna com tema Dark focada em usabilidade) e JavaScript assíncrono (Fetch API).
- **Backend:** Node.js com o framework Express para gerenciamento de rotas e regras de negócio.
- **Banco de Dados:** MySQL (Persistência de dados, relacionamentos e consultas estruturadas).
- **Comunicação:** APIs RESTful para integração robusta entre a interface e o servidor de banco de dados.
